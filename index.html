Before You Start: Enable Dark Mode

StudentLog currently doesn't have its own light/dark theme, and text can be very hard to read if Windows is set to light mode. Please switch Windows to dark mode first:

  1. Open Settings → Personalization → Colors.
  2. Set Default app mode to Dark.

Setup Instructions

  1. Install 7-Zip if you don't already have it — it's required to extract the download correctly.
  2. Download StudentLog.zip using the button above.
  3. Right-click the downloaded zip and choose 7-Zip → Extract Here (or Extract to "StudentLog\") to unzip it to a folder.
  4. Open the extracted Student Log folder and run setup - Shortcut.
  5. If prompted by Windows Defender or SmartScreen, click More info → Run anyway.
  6. The setup script will install the Windows App Runtime (if needed) and launch StudentLog automatically.
  7. For easy access later, move StudentLog - Shortcut out of the folder (e.g. to your Desktop) — use it to launch the app going forward.

System Requirements

  • Windows 10 or later (64-bit)
  • Internet connection for first-time setup (to install the Windows App Runtime, if needed)
